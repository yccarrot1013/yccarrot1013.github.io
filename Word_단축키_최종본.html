📊 고급 작업 (실무용)

🚀 꿀팁

  • Shift + F3 → 대소문자 변경
  • Ctrl + Enter → 페이지 나누기
  • Ctrl + Shift + C / V → 서식 복사 / 붙여넣기
  • Alt 키 → 메뉴 단축키 표시
  • Ctrl + Home / End → 문서 맨 위/아래 이동
  • Ctrl + ← / → → 단어 단위 이동
  • Ctrl + Shift + ← / → → 단어 선택
  • Ctrl + K → 하이퍼링크 삽입
  • Alt + Ctrl + F → 각주 삽입
